FIRST LOOK

海外ブランドは「人気」だけで仕入れない。

知名度が高いブランドほど仕入れ値も上がりやすく、薄利になりがちです。型番、ケースサイズ、保証書、人気ラインを確認し、説明できる根拠がある個体だけを見ます。

BRAND NOTES

ブランド別の見るポイント

ROLEX / ロレックス

HIGH VALUE

誰もが知る高級時計ブランド。人気は強いものの、仕入れ価格が高くなりやすいため、利益幅を冷静に見る必要があります。

確認ポイント
  • 型番が示すモデル・素材・装飾を読む
  • 仕入れ価格と販売手数料を先に計算
  • 真贋・保証書・付属品の確認を最優先

OMEGA / オメガ

GLOBAL

ROLEX同様、世界的な知名度があるスイスブランド。モデル名の認知が高く、検索需要を拾いやすいのが特徴です。

確認ポイント
  • Seamaster、Speedmasterなどライン名を明記
  • 保証カードに記載された型番を確認
  • 偽物対策として付属品と写真の整合性を見る

TAG Heuer / タグホイヤー

SPORTS

クロノグラフやスポーツウォッチで知られるブランド。比較的安価に仕入れられる中古モデルもあり、初心者が見やすい領域です。

確認ポイント
  • Formula 1、Aquaracerなどシリーズで整理
  • ケース幅でメンズ/ユニセックス/レディースを確認
  • 色違い、ブレス違いの相場差を見る

Hamilton / ハミルトン

CLASSIC

アメリカ発祥で現在はスイス系グループ傘下。ミリタリーや鉄道時計の背景があり、ストーリーを添えやすいブランドです。

確認ポイント
  • Hから始まる型番を裏蓋で確認
  • カーキ、ジャズマスターなどラインを分ける
  • ケース素材、文字盤色、サイズを説明に入れる
